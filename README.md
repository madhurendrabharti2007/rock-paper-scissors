# rock-paper-scissors
Simple Rock Paper Scissors game using Python
## Objective
- To understand basic Python programming concepts
- To practice decision-making using if-else statements
- To implement loops for continuous gameplay
- To build a simple interactive game
## Feature
- User vs Computer gameplay
- Continuous game using while loop
- Decision making using if-else
- Random choice generation by computer
- Exit option for user
